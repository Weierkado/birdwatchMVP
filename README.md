# 观鸟模拟器 · 纯文字 MVP

这是一个使用原生 HTML、CSS、JavaScript 制作的纯文字观鸟模拟器验证版。

## 运行方式

使用 VSCode Live Server 打开 `index.html`。

不需要安装 npm，不需要构建步骤。

## 当前玩法

- 点击开始游戏进入 30 回合观鸟。
- 通过观察、转向、静听、等待寻找鸟类。
- 当前鸟点无明显线索时，可以倾听远处的声音并切换鸟点。
- 不同鸟点会按各自的鸟种权重生成活动鸟。
- 发现鸟后进入拍照状态。
- 选择按下快门、再等一等或放弃拍摄。
- 拍到照片后抽取卡牌并写入图鉴。
- 回合结束后查看本局结算。
