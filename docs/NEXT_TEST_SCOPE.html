下一版测试范围 · 交接文档

核心方向：验证「所见即所得」对焦系统 + 鸟种差异化体验
功能数量：4 项 主要新增：对焦玩法 文案：维持上版品质，不做打磨

一、这次测试要验证什么

上一版测试验证了文字玩法的基础接受度和抽卡模型。这一版引入了对焦交互系统，核心需要验证的概念是：

所见即所得：玩家在取景框里追踪的移动 badge 上实时显示行为状态（寻常 / 有趣 / 精彩）。按下快门时 badge 显示什么状态，就拍到什么稀有度的卡牌——没有随机池，不抽奖，所见即所得。

这个概念是否被玩家理解，直接决定了对焦玩法能否成立。

同时验证鸟种差异化是否有感知（6种鸟的对焦手感是否被玩家区分），以及若干上版遗留的认知问题（SD卡误解等）是否已消除。

二、本版功能范围

1
对焦玩法 + 对焦词缀
FOCUS_AFFIX_DESIGN.html
在拍照阶段引入对焦交互：屏幕下方出现对焦区域，行为状态 badge 按鸟类运动模式持续移动。玩家需要在 badge 进入绿色对焦框时按下快门。

对焦词缀（两档）：badge 在框内点击 = 正常；框外点击 = 失焦。失焦照片积分 -1（下限 0），仍然解锁图鉴词条。失焦照片在卡牌稀有度 badge 后显示「失焦」标签。

双轴判定：badge 位置 → 对焦词缀（失焦/正常）；badge 状态 → 卡牌稀有度（寻常/有趣/精彩）。两轴均无概率因素，全由玩家操作决定。
2
初见系统 + 加新优化
FIRST_ENCOUNTER_DESIGN.html
玩家第一次遭遇某种鸟时触发专属初见文本，呈现该鸟「内行秒认 × 外行觉得不可思议」的核心特征描述（想象力钩子）。

加新流程优化：首次拍到某鸟时的「加入图鉴」仪式感提升，让玩家清晰感知到「我认识了一只新鸟」。
3
鸟种替换（6种）
SPECIES_DESIGN.html
替换上版5种同质化鸟种，引入差异明显的新组合：

翠鸟（高难度 / 高频抖动）· 麻雀（中难度 / 不规律漫游）· 红嘴蓝鹊（中难度 / 有章法扫掠）· 乌鸫（中高难度 / 警觉停顿）· 夜鹭（低难度 / 近乎静止）· 鸳鸯（低难度 / 稀有遭遇）

每种鸟有独立的运动参数、行为序列权重和初见文本，设计意图是让玩家在多次遭遇后形成「这种鸟要这样对待」的策略认知。
4
相机电量 UI（SD卡换皮）
BATTERY_UI_DESIGN.html
纯换皮，机制不变。将「SD卡」替换为「电量」概念，UI 改为电池图形 + 百分比显示（满格 100%，每拍一张递减）。

上版测试发现部分玩家不知道 SD 卡是什么，甚至产生「要把 SD 卡拍满」的错误目标认知。电量是玩家高度熟悉的有限资源概念，不需要额外解释。

颜色阈值：>30% 绿色 · 11–30% 黄色 · ≤10% 红色闪烁 · 0% 空壳静止。
本版文案策略：维持上版文案品质，仅修正会造成机制误解的表述（如剩余 SD 卡相关文本），不做文字质感的追求性打磨。文案品质提升作为独立专题留待后续版本，避免混淆本次机制测试的信号。

三、本版明确不做

  • 文案质感提升（初见文本、行为描述等文字打磨）
  • 局外养成系统（图鉴积分消耗、相机升级、行动力升级）
  • 珍贵稀有度触发（需要特定天气 × 时间 × 地点的环境条件触发）
  • 焦内序列动态切换（参数已设计完毕，功能实现延后）
  • 地图 / 世界探索系统
  • 主线叙事内容

四、关键验证指标

所见即所得是否被理解？
按快门前询问玩家：「你现在按下去会拍到什么？」——能否正确说出当前 badge 状态对应的稀有度
大多数玩家能在 1–2 局后建立正确预期，不再认为结果是随机的
失焦归因是否正确？
首次失焦后询问：「刚才为什么失焦了？」——观察玩家归因方向
归因为「我没点准 / 没追到」而非「游戏随机了」
初见钩子效果如何？
首次遭遇某鸟后询问：「刚才那只鸟有什么特别的地方？」
能用自己的话复述出该鸟的某个反常识特征（说明文字触发了认知）
鸟种差异化是否有感知？
遭遇多种鸟后询问：「哪只鸟最难拍？哪只最容易？」
能主动区分至少 2 种鸟的对焦手感差异，且方向符合设计意图（翠鸟难、夜鹭易）
电量误解是否消除？
观察玩家行为：是否出现「先把电量打空再看」的操作；或直接询问「电量用完了之后呢？」
不再出现「要把电量用完」的错误目标；理解电量 = 拍摄机会上限
对焦难度曲线是否合理？
埋点：失焦率 by 鸟种（失焦张数 / 该鸟总拍摄张数）
失焦率排序：翠鸟 > 乌鸫 > 麻雀 > 红嘴蓝鹊 > 夜鹭 ≈ 鸳鸯
以下任一情况出现，视为对应功能验证失败，需在下版优先修复：

· 超过半数玩家认为卡牌稀有度是按快门后随机抽取的 → 所见即所得机制失效
· 仍然出现「把电量打空」的策略行为 → 电量换皮未能消除误解
· 玩家无法区分任意两种鸟的对焦手感 → 鸟种差异化失败
