一、架构变更：focus.green 迁移

对焦绿框大小是相机精度参数，与鸟种无关，从 focusConfig.js 移出，改为全局配置：

// config.js 新增
CAMERA_FOCUS_GREEN: 60,     // 对焦正常判定半径（px），当前为常量
// 养成升级接口预留：未来由 gameState.cameraLevel 控制此值

// focusConfig.js 各条目中删除：
focus: { green: 60, perfect: 20 } // ❌ 整个 focus 字段移除

focusEngine.js 中 getFocusAffixFromPosition() 改为从 config.CAMERA_FOCUS_GREEN 读取判定半径，不再从 focusConfig 读取。

相机升级系统预留：养成树里的「升级相机」选项将来通过修改 gameState.cameraLevel.greenRadius 覆盖默认值，不需要改动 focusConfig.js。本次测试版使用固定值 60px，主线开发时接入。

六种鸟覆盖三个对焦难度档次、两个 allowJump 模式、从高频到低频的不同行为节奏。玩家从麻雀（熟悉、易上手）出发，逐步积累对不同鸟类行为模式的感知，翠鸟和乌鸫提供挑战上限，鸳鸯和夜鹭提供放松窗口，红嘴蓝鹊提供「可学习的中等挑战」。
翠鸟 kingfisher
水边 清晨最活跃 出现权重：中 对焦难度：高 allowJump：✓ 可跳跃
它比你想象的小——全身蓝绿，但只有拇指那么宽。你注意到它停在水面上方的枯枝时，它已经走了：一道蓝光消失进水里，留下一圈涟漪。你才意识到刚才那不是光线反射。

行为序列中精彩稀有且短暂；对焦高频抖动，REMARKABLE 时横向扫掠，窗口极窄；allowJump 开启——精彩可以毫无预告地出现。

麻雀 sparrow
全域 全时段 出现权重：高 对焦难度：中 allowJump：✓ 可跳跃
你以为你认识麻雀。但你说不出它背上的斑纹是什么样的——棕色与黑色交织，每根羽毛单独描绘，没有两只完全相同的。你只是从来没仔细看过。

行为节奏均衡，不规律——时间分布上随时能出精彩。对焦中等难度，wander 运动夹杂不定期停顿，节奏感来自不可预测而非速度。allowJump 开启体现其活泼特性。

红嘴蓝鹊 red_billed_magpie
林缘 / 山地 上午 出现权重：中 对焦难度：中 allowJump：✗ 顺序切换
尾巴比身体长两倍。落下时先落尾巴——整个过程像一面蓝色的旗帜慢慢展开，然后它转过头，你看到那抹橙红色的嘴。杭州山边的林子里并不难遇见它，但第一次遇见时你还是会说不出话来。

INTERESTING 状态丰富，REMARKABLE 出现有章法——不突兀，可预判，让玩家有「我看懂它了」的感觉。运动节奏有扫掠感（呼应长尾巴），allowJump 关闭，节奏稳定可学习。

鸳鸯 mandarin_duck
水边 / 静水 全时段 出现权重：低（稀有） 对焦难度：低 allowJump：✗ 顺序切换
雄性脸颊有两缕白色胡须，橙色帆羽竖起来像船帆。这些装饰唯一的功能是让雌性选择它——完全没有实际用途。它带着这一身炫耀的行头在水面上游，慢得像个展示品。

低出现权重制造稀有感；一旦遭遇，行为慢而稳定，对焦极为宽松——鸳鸯的张力来自「难得一见」，拍摄本身应当是奖励性的体验。

乌鸫 blackbird
城市公园 / 灌丛 清晨 出现权重：中偏高 对焦难度：中偏高 allowJump：✗ 顺序切换
全身黑色，但喙是橙黄色——像涂了一圈唇膏。叫声能模仿至少十种其他鸟的鸣叫。这就是为什么你在林子里听到很多种声音，但找来找去只找到了一只乌鸫。

警觉性高，移动间夹杂频繁定格停顿（头抬起观察环境）；对焦中偏难，stutter 概率高，突然静止打断节奏感是其核心手感。
